# Patch 13 - Demo Kit (README + demo script)

## Objetivo
Completar un kit de demo comercial listo para presentar y grabar, con alcance claro, flujo recomendado, features a remarcar y activación de demo tools.

## Cambios realizados

1. **README convertido en Demo Kit**
   - Qué es y qué no es la demo (alcance).
   - Flujo sugerido de 60–120 segundos.
   - Lista de features clave para el pitch.
   - Sección “Modo demo tools” (`?demo=1` y `NEXT_PUBLIC_DEMO_TOOLS`).
   - Quickstart local para levantar demo y activar herramientas.
   - Instrucciones para exportar/aplicar patch con `git apply`.

2. **Guion de demo y checklist de grabación**
   - Nuevo archivo `docs/DEMO_SCRIPT.md`.
   - Bloques temporizados + narrativa sugerida.
   - Checklist técnico (resolución/FPS/ratio).
   - Páginas mínimas a mostrar y CTA final.

## Resultado esperado
- Demo más consistente para ventas y grabaciones.
- Menos fricción para habilitar `/adminv1` en contexto comercial.
- Entrega reutilizable vía patch.
