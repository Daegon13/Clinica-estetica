# VetCare service icons

This folder is intended to contain the **8 individual service icons** exported from `public/brand/services-icons.webp`.

## Expected filenames (WebP recommended)

- `consulta.webp` (stethoscope + paw)
- `vacunacion.webp` (clipboard + syringe)
- `desparasitacion.webp` (pill + check)
- `laboratorio.webp` (microscope)
- `cirugia.webp` (scalpel + paw + heartbeat)
- `internacion.webp` (resting pet / hospitalization)
- `grooming.webp` (bath)
- `control.webp` (calendar + check)

> Tip: export each icon to **512×512** (or 1024×1024) with a bit of padding so it doesn't feel cramped.
